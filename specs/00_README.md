## 🎯 무엇을 만드나
**DEEPWRITE** = 상급 영어 학습자를 위한 **AI 작문 트레이너**.
사용자가 영어 문장/글을 쓰면, *통역사가 옆에서 고쳐주듯* 5차원으로 분석하고,
네이티브 수준으로 다시 써주며, **왜 그렇게 써야 하는지**까지 설명한다.
(자매 앱 DEEPREAD가 "읽기"라면, DEEPWRITE는 "쓰기"다.)

## ⚙️ 작업 원칙 (반드시 지킬 것)
1. **한 티켓 = 한 단위.** 티켓 하나 끝낼 때마다 멈추고 사용자에게 "T0X 완료, 확인해주세요" 보고.
2. **모든 외부 입력은 Zod로 검증.** 타입 안전 최우선.
3. **DB 변경은 마이그레이션 파일로.** `supabase/migrations/`에 timestamp 파일 생성.
4. **모든 LLM 호출은 비용·캐싱 고려.** 04 문서의 모델 분리 규칙을 따를 것.
5. **사양이 모호하면 임의 판단 금지 → 사용자에게 질문.**
6. **에러로 막히면** 에러 메시지 원문 + 시도한 내용을 사용자에게 정리해서 보고.
7. **자주 배포.** 의미 있는 단위마다 Vercel preview로 사용자가 폰에서 확인할 수 있게.
8. **deepread 코드를 import 하지 말 것.** 디자인은 *복사*로만 맞춘다 (02 문서 참고). 두 앱은 독립.

## 🧱 기술 스택 (deepread와 동일)
- **Next.js 14 (App Router) + TypeScript**
- **Supabase** (Auth + Postgres + RLS)
- **Tailwind CSS** (deepread 디자인 토큰 복사)
- **Anthropic Claude API** (분석·리라이트·역번역 엔진)
- **Vercel** 배포

## ✅ 1차 출시(MVP) 정의
다음이 동작하면 MVP 완료:
1. 로그인 → 온보딩 진단 작문
2. 작문 에디터에서 글 입력 → **목표 톤 선택(격식/중립/구어)** → 5차원 진단 + 네이티브 리라이트 + Diff + "왜" 설명
3. 작성 중 **라이브 힌트**(콜로케이션 + 비교단어 카드)
4. **3톤 동시 변환** 비교
5. **역번역 트레이닝** 1세트
6. **개인 약점 리포트** 누적
7. 결제(토스페이먼츠 or Stripe) 연동된 Free/Pro 게이트
