# 01. PRD — DEEPWRITE 제품 요구사항

## 1. 제품 개요
**한 줄 정의:** "내가 쓴 영어를, 통역사가 옆에서 고쳐주듯 5차원으로 분석하고 — *왜* 그렇게 써야 하는지까지 알려주는 작문 트레이너."

**해결하는 문제:** 상급 학습자는 *읽으면* 이해하지만 *쓰면* 어색해진다. 문법은 맞아도 콜로케이션·레지스터·뉘앙스가 틀리고, 첨삭을 받아도 "왜"를 몰라 또 틀린다.

**솔루션:** 내 글 ↔ 네이티브 글을 **나란히 비교(격차 인식)** + 변경 이유·규칙·빈도 설명 + 약점 누적 추적.

## 2. 교육 엔진 (모든 기능의 근거)
1. **산출 강제** — 수동 첨삭이 아니라 직접 생산.
2. **격차 인식** ⭐ — 내 문장 ↔ 네이티브 문장 나란히 + 차이 이유.
3. **즉각·구체 피드백** — "어색"이 아니라 이유+규칙+빈도.
4. **의도적 연습** — 약점에 집중한 맞춤 드릴.
5. **역번역** — 통역사 무기. 의도 보존 검증.

## 3. 타겟 / 페르소나
- **통대생 지원이(24):** 시험형 작문(요약·에세이·역번역) + 객관 점수.
- **외국계 직장인 현우(33):** 이메일·보고서 + 톤 조절 + 빠른 리라이트.
- **유학·논문생 서연(28):** 학술 레지스터·헤지·논리 응집성.
공통 페인: "고쳐는 주는데 다음에 또 틀린다."

## 4. 기능 목록

### 4.1 [MVP] 5차원 작문 진단 + 네이티브 리라이트 ⭐ 심장
사용자 글 입력 → 5차원 진단(어휘/콜로케이션/문장구조/응집/톤) → 네이티브 리라이트 → 단어 단위 Diff(추가/삭제/교체) → 각 변경의 **이유+규칙+빈도** 설명. (프롬프트: 04 §1)

### 4.2 [MVP] 멀티 톤 작문 엔진 ⭐ (신규 요구사항 1)
- **작문 전 목표 톤 선택:** `격식체 / 중립 / 구어`. 선택한 톤이 5차원 진단의 채점 기준이 됨 (격식체에선 축약형 감점, 구어에선 과도한 격식어 완화 권장 등).
- **작문 후 3톤 동시 변환:** 같은 내용을 세 톤으로 펼쳐 비교 + **무엇이 톤을 만드는가**(동사·연결어·명사화·신호어) 설명.
- **톤 일관성 검사:** 한 글 안에서 톤이 튀는 문장 경고.
(프롬프트: 04 §2)

### 4.3 [MVP] 라이브 작문 어시스트 ⭐ (신규 요구사항 2)
작성 중 실시간 힌트:
- **콜로케이션 힌트:** 핵심어 입력 시 자연스러운 조합 + **빈도 막대 + 격식 수준 + Konglish 경고**. 현재 선택 톤에 맞춰 정렬.
- **비교 단어 카드:** 단어 선택 시 → 비슷한 뉘앙스 / 차이(격식·강도·맥락) / **반의어** / 자주 가는 조합. (DEEPREAD 유의어 정밀비교 자산 재사용)
- **힌트 노출 모드 3종:** `즉시 힌트` / `시도 후 힌트(기본)` / `힌트 끄기(시험모드)` — 베껴쓰기 방지, 격차 인식 엔진 보호.
(프롬프트: 04 §3)

### 4.4 [MVP] 역번역 트레이닝
한국어 의도 → 영어 작성 → AI가 영어를 한국어로 역번역 → 원래 의도와의 차이 표시 → **의도 일치도(%)** + 네이티브 모범. (프롬프트: 04 §4)

### 4.5 [MVP] 개인 약점 패턴 추적
모든 첨삭의 오류를 유형별로 누적(관사/콜로케이션/시제/전치사/run-on 등) → 랭킹·추이 차트 → 맞춤 드릴 추천.

### 4.6 [MVP] 온보딩 진단
가입 직후 짧은 작문 3개 → 5차원 베이스라인 + 추정 CEFR + 최약점 진단.

### 4.7 [Phase 2] 장르별 트랙
이메일 / 보고서·제안서 / 에세이·사설 / 학술 / 요약·역번역. 각 장르 루브릭 + 분야별 코퍼스(비즈니스/외교/IT/법률/의료).

### 4.8 [Phase 2] 단계별 빌드업 + 개작 사이클
문장→문장결합→단락→전문→**개작(1차 vs 2차 향상도)**.

### 4.9 [Phase 2] DEEPREAD 연동
deepread에서 읽은 기사 → "이 주제로 써보기" 과제 / 저장 표현 → "이 표현 써보기" 미션. (앱 간 딥링크 + 공유 계정)

### 4.10 [Phase 3] 챌린지·리더보드 / 점수 추이·CEFR / B2B 플랜

## 5. 수익화
- Free: 1일 작문 3개, 진단 요약본, 약점 미리보기.
- **Basic ₩9,900:** 작문 무제한, 5차원 전체, 약점 추적.
- **Pro ₩19,900 (메인):** 역번역, 멀티톤 전체, 라이브 힌트 전체, 장르 트랙, 개작, deepread 연동.
- **Master ₩39,900:** 학술/통대 특화, 심층 피드백, CEFR 리포트.
- **Business ₩15,000/인:** 회사 문체 가이드, 관리자 대시보드.
- MVP 게이트: 라이브 힌트 '비교 단어 카드'와 '역번역'을 Pro 전용으로 게이팅.

## 6. 비기능 요구
- 진단 응답 P95 < 8초 (스트리밍으로 체감 단축).
- LLM 비용: 캐싱 + 모델 분리로 글 1건당 목표 단가 관리.
- 모바일 우선 반응형. 다크모드 deepread와 동일.
- 개인정보: 사용자 글은 본인만 열람(RLS). 학습 통계는 익명 집계만.
